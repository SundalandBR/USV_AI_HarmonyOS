## 亮点功能

- 基于Yolov8-seg的河道障碍检测分割
- 基于Socket的视频推流服务
- 基于Mavlink协议的PX4通信

## 项目地址

- [USV_AI_HarmonyOS](https://github.com/SundalandBR/USV_AI_HarmonyOS)

## 使用说明

- 机载电脑端
  1. 进入该项目的`/bin/release`文件夹
  2. 根据需求填写`config.ini`配置文件
  3. 运行`python main.py`
- HarmonyOS端
